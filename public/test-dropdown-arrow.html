🎨 样式特性

🔧 技术实现

使用SVG图标作为背景图，实现自定义下拉箭头：

/* 主要样式 */ .product-form-select { appearance: none; -webkit-appearance: none; -moz-appearance: none; background-image: url('data:image/svg+xml;charset=US-ASCII,<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 12 8" fill="none"><path d="M1 1L6 6L11 1" stroke="%23ffffff" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"/></svg>'); background-repeat: no-repeat; background-position: right 15px center; background-size: 16px 12px; padding-right: 45px; cursor: pointer; } /* 悬停效果 */ .product-form-select:hover { background-image: url('...stroke="%23f39c12"...'); }

核心改进点：

  • 移除默认外观：使用 appearance: none 移除浏览器默认样式
  • SVG箭头图标：使用内联SVG作为背景图，确保清晰度
  • 颜色状态切换：正常、悬停、聚焦状态使用不同颜色
  • 跨浏览器兼容：支持Chrome、Firefox、Safari等主流浏览器
  • 响应式设计：在不同设备和屏幕尺寸下保持良好效果
